# 🏦 Banking System in C++

A simple **console-based Banking Management System** built in C++.  
It supports role-based access for bankers and customers, enabling secure account management, transactions, and data persistence using object-oriented design principles.  


## ✨ Features
- 👨‍💼 **Banker Login** – Secure login for bank administrators  
- ➕ **Create Customer** – Add new customers with ID and password  
- 📜 **View Customers** – Display all registered customers  
- 🚪 **Customer Login** – Authentication for individual users  
- 💰 **Deposit Money** – Add funds to customer account  
- 💸 **Withdraw Money** – Withdraw funds with balance validation  
- 📊 **Check Balance** – View current account balance  
- 🧾 **Transaction History** – Track deposits and withdrawals  
- 💾 **File Handling** – Persistent storage of customer data  
- 🧩 **Modular Design** – Clean separation of classes and logic  


## 🧠 OOP Concepts Used
- **Encapsulation** → Data hiding using private class members and public methods  
- **Inheritance** → Shared functionality across banker and customer classes  
- **Polymorphism** → Flexible function behavior via method overriding / interfaces  
- **Abstraction** → Simplified interface for complex banking operations  


## 🛡️ Security & Validation
- **Authentication** → Login system with ID and password verification  
- **Input Validation** → Prevents invalid or empty inputs  
- **Balance Checks** → Prevents overdraft during withdrawal  
- **Unique Customer IDs** → Avoids duplication  
- **Data Integrity** → Ensured using structured file storage  


## ⚙️ Tech Stack
- **Language** → C++  
- **Concepts** → OOP, File Handling, Data Structures  
- **Environment** → Linux / GCC / G++  


## 🚀 How to Run
g++ *.cpp  
./a.out
